## Description

To use this template, copy the following code and fill in the
appropriate fields.

## Syntax

    {{infobox event
     | title         =
     | image         = [e.g. "Example.jpg"]
     | imagecaption  =
     | performers  =
     | date          =
     | location      =
    }}

## Samples

    {{infobox event
     | title         = Flower du Jour
     | image         = Example.jpg
     | performers    = Awesome Ones
     | date          = All the time
     | location      = Everywhere
    }}
