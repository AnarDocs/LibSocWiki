## Description

To use this template, copy the following code and fill in the
appropriate fields.

## Syntax

    {{infobox box
     | title         =
     | image         = [e.g. "Example.jpg"]
     | author        =
     | illustrator   =
     | datePublished =
     | publisher     =
     | previous      =
     | next          =
    }}

## Sample output

    {{infobox book
     | title         = The Adventures of Flower
     | image         = Example.jpg
     | author        = Alice
     | illustrator   = Bob
     | datePublished = April 1928
     | publisher     = Arthur's
     | previous      = Book One
     | next          = Book Three
    }}
